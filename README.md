# DWGtoDXFConverter
Pipeline code for converting of DWG file to DXF file by using Teigha Software

# Paraemeters
TEIGHA_PATH: Location of .exe file \n
Input folder: Location of input folder \n
Output folder: Location of output folder \n
Output version: ACAD9, ACAD10, ACAD12, ACAD14, ACAD2000, ACAD2004, ACAD2007, ACAD20010, ACAD2013, ACAD2018 \n
Output file type: DWG, DXF, DXB \n
Recurse Input Folder: 0, 1 \n
Audit each file: 0, 1 \n
(Optional) Input files filter: *.DWG, *.DXF
